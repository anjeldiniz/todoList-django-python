
# Clone Repository
```bash
$ git clone git@github.com:anjeldiniz/todoList-django-python.git
```
```bash
$ cd api_todo
```
# Install locally
```bash
$ pip install django
$ pip install djangorestframework
```
# Development
```bash
$ python manage.py runserver

http://127.0.0.1:8000/todo or http://127.0.0.1:8000/todo/pk

```
